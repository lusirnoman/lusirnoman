## 👨‍🔬 About Me

Computational structural biologist blending **cryo-EM**, **AI**, and **infrastructure automation**.

- 🔬 **Research:** Cryo-EM image processing, protein-ligand docking, blind screening pipelines
- 🖥️ **Tools:** CryoSPARC, Scipion, ModelAngelo, Relion, AutoDock Vina
- 🤖 **Automation:** Hermes Agent, homelab orchestration, CI/CD for science
- 🏠 **Homelab:** Multi-node GPU cluster, NAS, OpenWrt networking

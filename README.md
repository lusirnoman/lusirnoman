## 👨‍🔬 About Me

<img align="right" width="160" src="https://github.com/lusirnoman/lusirnoman/raw/main/assets/raven-logo.svg" />

**Lusir (渡鸦)** — Computational structural biologist blending cryo-electron microscopy, artificial intelligence, and infrastructure automation to accelerate drug discovery.

- 🔬 **Research Focus** — Single-particle cryo-EM, protein-ligand docking, blind screening pipelines for antibody-antigen candidates
- 🧪 **Scientific Stack** — CryoSPARC, Scipion, Relion, ModelAngelo, Xmipp, EMAN2, AutoDock Vina, Topaz
- 🤖 **Automation** — Hermes Agent, multi-node GPU orchestration, automated cryo-EM processing pipelines
- 🏠 **Homelab** — Multi-node GPU cluster (work, splab), 25GbE NAS, NixOS, OpenWrt networking
- 🌐 **Edge Computing** — Cloudflare Workers ecosystem, serverless apps, edge proxies
